# IOC Checker for bulk IOC Checking

### Update your system
``sudo apt update``

### Install python
``sudo apt install python3 python3-pip``

### Install flask
``pip3 install Flask``

### Edit the script with your API
``sudo nano app.py``
### Save your script 
```
CTRL + X
Y
Enter
```
### Run the script
`` python3 app.py``

### Lastly
Copy paste the ``http`` running in port ``5000`` to your browser.
``http://127.0.0.1:5000``
